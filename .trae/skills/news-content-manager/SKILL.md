# News Content Manager

This skill helps manage and optimize news website content following journalistic best practices and web publishing standards.

## Capabilities

### 1. Article Creation
- Write news articles following inverted pyramid structure
- Create compelling headlines (under 10 words for web)
- Write SEO-optimized meta descriptions
- Generate appropriate tags and categories

### 2. Content Updates
- Update existing articles with latest information
- Refresh outdated content while preserving SEO value
- Add breaking news updates to existing stories
- Maintain content consistency across pages

### 3. Content Optimization
- Optimize for SEO (keywords, meta tags, structured data)
- Improve readability (short paragraphs, subheadings, bullet points)
- Add multimedia elements (images, videos, infographics)
- Ensure mobile-friendly formatting

### 4. News Categories Management
- Technology
- Business & Finance
- Health & Science
- Politics
- Sports
- Entertainment
- World News
- Local News

## Best Practices

### Writing Style
- Use KISS principle (Keep It Short and Simple)
- Follow AP Style guidelines
- Use active voice
- Include quotes from credible sources
- Fact-check all information

### Web Optimization
- Headlines: 6-10 words maximum
- Paragraphs: 2-3 sentences
- Subheadings every 300 words
- Include relevant keywords naturally
- Add internal and external links

### SEO Guidelines
- Meta title: 50-60 characters
- Meta description: 150-160 characters
- Use header tags (H1, H2, H3) properly
- Include alt text for images
- Implement schema markup for news articles

## Usage Examples

### Creating a News Article
```
User: "Create a news article about the latest tech announcement"
Action: Research latest news, write article following inverted pyramid structure, optimize for SEO
```

### Updating Content
```
User: "Update the homepage with today's breaking news"
Action: Identify outdated content, add new breaking news, maintain layout consistency
```

### Optimizing Existing Content
```
User: "Optimize this article for better search rankings"
Action: Add keywords, improve meta tags, enhance readability, add structured data
```

## Output Format

Always provide:
1. SEO-optimized headline
2. Meta description
3. Article content (inverted pyramid structure)
4. Suggested tags/categories
5. Internal linking suggestions
6. Image recommendations with alt text
